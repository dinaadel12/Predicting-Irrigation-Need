Predicting Irrigation Need

This project focuses on predicting irrigation requirements for crops using machine learning and deep learning techniques. The goal is to optimize water usage and support efficient agricultural decision-making.

 Problem:

Efficient irrigation is essential for improving crop yield and conserving water resources. However, variations in environmental conditions and low feature correlation make prediction challenging.

Approach:

Data preprocessing (handling missing values, scaling, and encoding).
Feature engineering to improve model performance.
Addressed class imbalance in the dataset.
Built a hybrid model using Deep Learning and LightGBM.
Evaluated model performance using accuracy and validation metrics.

Results:
Achieved 96.3% accuracy on Kaggle.
The model effectively handled unbalanced data and captured hidden patterns in low-correlation features.

Tools & Technologies:

Python, Pandas, NumPy, Scikit-learn ,TensorFlow, LightGBM ,XGboost
